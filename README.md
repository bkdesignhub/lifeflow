## ✨ About LifeFlow

LifeFlow is a modern personal management platform designed to help individuals organize their daily life in a simple and stress-free way.

Instead of using multiple apps for tasks, notes, reminders, and personal money tracking, LifeFlow combines everything into a single beautiful dashboard.

Built with a mobile-first approach, it feels like a native mobile application while working perfectly on desktop browsers.

# 🚀 Installation

Clone Repository

```bash
git clone https://github.com/bkdesignhub/lifeflow.git
```

Enter Project

```bash
cd lifeflow
```

Install Dependencies

```bash
composer install
```

Copy Environment

```bash
cp .env.example .env
```

Generate Key

```bash
php artisan key:generate
```

Run Migration

```bash
php artisan migrate
```

Start Server

```bash
php artisan serve
```
